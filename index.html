HTML Basics

1. HTML Structure

Every HTML document has a basic structure. The most important tags are:

  • <!DOCTYPE html>: Declares the document type and version.
  • <html>: The root element of an HTML page.
  • <head>: Contains metadata like the title, charset, and linked files.
  • <body>: Contains the content of the page, such as text, images, and links.

Example:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

2. Headings

HTML provides six levels of headings, from <h1> to <h6>, with <h1> being the most important.

Example:

<h1>This is a heading 1</h1>
<h2>This is a heading 2</h2>
<h3>This is a heading 3</h3>

3. Paragraphs

Paragraphs are defined with the <p> tag.

Example:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

4. Links

Links are created with the <a> (anchor) tag. The href attribute specifies the URL.

Example:

<a href="https://www.example.com">Visit Example</a>

5. Images

Images are embedded using the <img> tag. The src attribute defines the source, and alt provides alternative text.

Example:

<img src="image.jpg" alt="Description of image">

6. Lists

There are two types of lists in HTML:

  • Ordered lists (numbered) use the <ol> tag.
  • Unordered lists (bulleted) use the <ul> tag.

Example (Ordered List):

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

Example (Unordered List):

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

7. Tables

Tables are created using the <table> tag. The rows are defined with <tr>, headers with <th>, and data cells with <td>.

Example:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>30</td>
  </tr>
</table>

8. Forms

Forms allow user input and are defined using the <form> tag. Common form elements include text inputs, checkboxes, and buttons.

Example:

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

9. Divs and Spans

The <div> tag defines a block-level container, and the <span> tag defines an inline container.

Example:

<div>This is a div</div>
<span>This is a span</span>

10. Comments

HTML comments are added with <!-- comment --> and are ignored by browsers.

Example:

<!-- This is a comment -->
